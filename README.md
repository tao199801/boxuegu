## Android项目实战博学谷――WordPress建站APP

### 项目简介

- 此项目为`移动高级开发课程项目――博学谷
- 开发主题为――WordPress建站

### 功能特征

- 欢迎界面
- 课程界面、课程详情界面
- 习题界面、习题详情界面
- 个人账户模块、设置、修改密码

- `OS：Windows 10 x64`
- `Eclipse Jee 2019`
- `项目编译：Android 9.0`
- `ADT 23.0.6`

##### Android SDK:
